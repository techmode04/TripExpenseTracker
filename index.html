Cloud Auto-Sync

Data modify ya save karne par automatic background backup.

💡 Tip: Sabhi log apne mobile par apna personal Google Apps Script URL yahan dalkar apna data separate Google Sheet ke sath backup aur sync kar sakte hain!

🚨 IMPORTANT RULE FOR CORRECT SETUP (Sync failures se bachne ke liye):

Jab aap Apps Script deploy karein, toh yeh do options bilkul sahi select hone chahiye:

  • Execute as: Me (your-email@gmail.com)
  • Who has access: Anyone (Agar aapne "Anyone with Google Account" select kiya, toh request browser par fail ho jayegi).

Code for your Apps Script:

function doGet(e) {
  var scriptProp = PropertiesService.getScriptProperties();
  var data = scriptProp.getProperty("wanderbudget_db");
  return ContentService.createTextOutput(JSON.stringify({
    status: "success",
    data: data ? JSON.parse(data) : null
  })).setMimeType(ContentService.MimeType.JSON);
}

function doPost(e) {
  try {
    var requestData = JSON.parse(e.postData.contents);
    if (requestData.action === "sync") {
      var scriptProp = PropertiesService.getScriptProperties();
      scriptProp.setProperty("wanderbudget_db", JSON.stringify(requestData.data));
      return ContentService.createTextOutput(JSON.stringify({ status: "success" }))
        .setMimeType(ContentService.MimeType.JSON);
    }
  } catch(err) {
    return ContentService.createTextOutput(JSON.stringify({ status: "error", message: err.toString() }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}
